What this app does

AITA Stories Bot is a personal automation tool that reads publicly available posts from Reddit's r/AmITheAsshole subreddit, converts them into narrated short-form videos, and posts them to a single personal TikTok account via the official TikTok Content Posting API (video.publish scope).

Who uses it

This app is operated solely by its owner for their own personal TikTok account. It does not serve third-party users and does not collect, process, or share any third-party personal data.

TikTok API Integration

This app uses TikTok Login Kit for authentication and the Content Posting API (Direct Post) to publish videos. All API usage complies with TikTok's Developer Terms of Service and Platform Policies.
