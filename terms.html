1. Description of AITA Stories Bot

AITA Stories Bot is a personal automation application that reads publicly available posts from Reddit's r/AmITheAsshole community, converts them into narrated short-form video content, and publishes that content to a designated TikTok account using TikTok's official Content Posting API.

3. TikTok API Usage

AITA Stories Bot uses the TikTok Login Kit and Content Posting API (video.publish scope) in accordance with TikTok's Developer Terms of Service. The App only posts content to the account that has explicitly authorized it via TikTok's OAuth flow.
