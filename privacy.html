Privacy Policy

Last updated: May 6, 2025

This Privacy Policy describes how AITA Stories Bot ("the App", "we", "us") collects, uses, and protects information in connection with your use of our application.

1. About AITA Stories Bot

AITA Stories Bot is a personal automation tool that uses the TikTok Content Posting API (video.publish scope) and Login Kit to authenticate and post videos to a single personal TikTok account. The App is not a public service and does not have third-party end users.

2. Information We Collect

When you authorize AITA Stories Bot via TikTok, we may receive:

  • Your TikTok User ID (open_id) — used solely to identify the account to post to
  • An OAuth access token — used to authenticate API calls to post videos

We do not collect passwords, payment information, email addresses, or any other personal data beyond what TikTok's API provides during the OAuth authorization process.

3. How We Use Your Information

AITA Stories Bot uses collected information only to:

  • Authenticate your TikTok account via OAuth 2.0 (Login Kit)
  • Post automated video content to your authorized TikTok account (video.publish)
  • Maintain the functionality and operation of the App

We do not sell, trade, rent, or share your information with any third parties.

4. TikTok API Scopes Used

AITA Stories Bot requests the following TikTok API scopes:

  • user.info.basic — to retrieve basic profile info for authentication
  • video.publish — to post videos directly to the authorized TikTok account

5. Data Storage and Security

Your TikTok access token and user ID are stored locally on the server running AITA Stories Bot. We use industry-standard security practices to protect this data. Tokens are never shared with unauthorized third parties and are only used for the purposes described above.

6. Data Retention

We retain your access token and user ID only as long as the App is active. You may revoke AITA Stories Bot's access at any time by going to TikTok Settings → Security → Authorized Apps and revoking access there.

7. Third-Party Services

AITA Stories Bot sources content from Reddit's public API. No personal user data is transmitted to Reddit. The App uses TikTok's official API — please refer to TikTok's Privacy Policy for information on how TikTok handles data.

8. Children's Privacy

AITA Stories Bot is not directed at children under 13. We do not knowingly collect personal information from children under 13. If we become aware that such information has been collected, we will delete it promptly.

9. Your Rights

You have the right to:

  • Access the data AITA Stories Bot holds about you
  • Request deletion of your data by contacting us
  • Revoke the App's access at any time via TikTok Settings → Security → Authorized Apps

10. Changes to This Policy

We may update this Privacy Policy from time to time. We will update the date at the top of this page to reflect changes. Continued use of AITA Stories Bot after changes constitutes acceptance of the updated policy.
